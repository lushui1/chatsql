## 如何新增方言适配规则

在 System Prompt 的方言规则段中添加：

```python
def _build_dialect_rules(self, dialect: str) -> str:
    rules = {
        "mysql": """
- 使用反引号 ` 引用标识符
- 日期: DATE_FORMAT / DATE_SUB / NOW()
- 分页: LIMIT offset, count
""",
        "yourdb": """
- 使用双引号 " 引用标识符
- 日期: YOUR_DATE_FORMAT / YOUR_DATE_SUB
- 分页: FETCH NEXT n ROWS ONLY
""",
    }
    return rules.get(dialect, rules["mysql"])
```
