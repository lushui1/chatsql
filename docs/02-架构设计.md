## 数据流：用户提问到结果展示

```
用户输入: "最近7天每个分拣中心的订单量趋势"
    │
    ▼
┌──────────────────────────────┐
│ 1. API 接收 (POST /v1/responses) │
│    解析 input，创建 session     │
└──────────────┬───────────────┘
               │
               ▼
┌──────────────────────────────┐
│ 2. ResponsesService 组装请求    │
│    - System Prompt（元数据+方言）│
│    - 学习经验注入               │
│    - 工具定义（4个内置工具）     │
│    - 用户消息                   │
└──────────────┬───────────────┘
               │
               ▼
┌──────────────────────────────┐
│ 3. LLM 流式响应 (SSE)          │
│    - function_call: planning   │
│    - function_call: chart      │
│    - output_text: "分析结果..." │
└──────────────┬───────────────┘
               │
               ▼
┌──────────────────────────────┐
│ 4. 工具执行                    │
│    planning → 返回分析步骤      │
│    chart → 返回图表配置         │
│    clarification → 返回追问     │
└──────────────┬───────────────┘
               │
               ▼
┌──────────────────────────────┐
│ 5. 前端渲染                    │
│    - 用户消息（右侧蓝色气泡）   │
│    - 工具卡片（可折叠展开）     │
│    - 图表（ECharts 渲染）       │
│    - 助手文本（左侧灰色气泡）   │
└──────────────────────────────┘
```

## 后端 DDD 四层

| 层 | 职责 | 目录 |
|----|------|------|
| **Presentation** | HTTP 路由、请求/响应格式 | `presentation/http/routes/` |
| **Application** | 业务逻辑编排、工具执行、LLM 调用 | `application/` |
| **Domain** | 核心领域模型、业务规则 | `domain/` |
| **Infrastructure** | 数据库 ORM、文件存储 | `infrastructure/` |

依赖方向：Presentation → Application → Domain ← Infrastructure

## 前端组件结构

```
App.vue                    # 主页面（侧边栏 + 聊天区 + 输入框）
├── ChartCard.vue          # 图表卡片（ECharts 封装）
└── SettingsView.vue       # 设置面板
    ├── 数据源 Tab          # 数据源 CRUD + 测试连接
    ├── 大模型 Tab          # LLM Provider 配置
    └── 系统 Tab            # 健康检查 + 版本信息
```
