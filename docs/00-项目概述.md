# 00 — 项目概述

## ChatSQL 是什么

ChatSQL 是一个开源的智能问数框架，让用户通过自然语言与数据库对话，自动生成 SQL、执行查询、返回结果并可视化展示。

**一句话定位**：把自然语言翻译成 SQL，把数据翻译成图表。

## 解决什么问题

| 痛点 | ChatSQL 的解法 |
|------|---------------|
| 不会写 SQL 的人想查数据 | 自然语言直接问，Agent 自动生成 SQL |
| 数据库方言多，写不对 | 自动适配 MySQL/ClickHouse/Hive 等 8 种方言 |
| 查完数据还要做图表 | 内置图表工具，柱/线/饼/表格自动选 |
| 同一个问题反复问 | 学习系统记住高频模式，越用越准 |
| 多个库数据要联合看 | 跨数据库查询，应用层自动拼接结果 |

## 核心能力

| 能力 | 说明 |
|------|------|
| 🗣️ 自然语言问数 | 用户说人话，Agent 翻译成 SQL |
| 🗄️ 8 种数据源 | DuckDB / MySQL / PostgreSQL / ClickHouse / Doris / Hive / Presto / Spark |
| 🤖 9 种 LLM | OpenAI / Anthropic / Google / DashScope / 智谱 / Moonshot / DeepSeek / Ollama / Custom |
| 📊 自动图表 | 根据数据特征自动选择柱状图/折线图/饼图/表格 |
| 📋 分析规划 | 复杂问题先拆解步骤再执行 |
| 🔀 跨库查询 | 多个数据源的结果应用层合并 |
| 📚 学习系统 | 从对话中提炼经验，越用越懂业务 |
| ⚡ 双模式 | fast（快速回答）/ think（深度分析） |
| 🎨 亮暗主题 | 前端支持浅色/深色主题切换 |

## 技术栈

| 层 | 技术 |
|----|------|
| 前端 | Vue 3 + TypeScript + Vite + ECharts |
| 后端 | Python 3.11+ + FastAPI + SQLAlchemy |
| 数据库 | SQLite（会话存储）+ 8 种外部数据源 |
| LLM | OpenAI 兼容协议（支持 9 种 Provider） |
| 部署 | Docker Compose / 本地双进程 |
